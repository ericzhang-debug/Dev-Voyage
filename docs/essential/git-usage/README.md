# 版本控制基础

::: tip 写在前面
欢迎来到 Git 的世界！如果你曾经因为误删代码而崩溃，或者文件夹里堆满了 `最终版_v10.doc`，那么这个教程就是为你准备的。我们将带你掌握程序员必备的“后悔药”——Git。
:::

## 🎯 这个教程适合谁？

- 经常手滑删文件、改坏代码的“手残党”
- 想要进入互联网行业，必须掌握 Git 的准程序员
- 对“开源”、“GitHub”感兴趣，想参与其中的开发者

## 📚 你将学到什么？

通过本教程，你将：

1. **理解版本控制** —— 明白它为什么被称为代码界的“时光机”
2. **掌握核心操作** —— 学会 `add`, `commit`, `push` 等日常指令
3. **学会分支管理** —— 在平行宇宙中开发功能，互不干扰
4. **玩转 GitHub** —— 将代码托管到云端，与全世界分享
